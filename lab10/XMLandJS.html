  1. What is XML?

    2. XML stands for extensible markup language. It was developed without the use of predefined tasks and the user can use user generated tags to define the data in the language.

  2. How is XML used?

    3. XML is primarily used to send and receive data. HTML is used with a data presentation in mind. XML is used primarily to send and receive data. XML is not used to display data and is actually a simplified method to carry data.

  4. What is JavaScript?

    5. Is a programming language that is used to add interactive displays and effects on a webpage. It is used to display objects and programming that is processed and displayed within a web browser.

  5. How is JavaScript related to HTML and CSS?

    6. JavaScript is related to HTML and CSS in the fact that JavaScript is embeded within these files for use on the webpage. All of these files are use client side processing to display the webpage. This means that the web browser processes these files and displays them

  6. What are some common JavaScript elements, properties, methods, and functions?

    7. A common JavaScript elements are: "string" which returns a value by name. JavaScript properties are a an abstract concept and adding a property to an object can be as simple as applying a "english" tag to an object that contains english text. A common method is the "toUpperCase()" method which converts an object to all upper case text. A function is JavaScript is a code that is deigned to perform a task. Functions can be invoked and reused. A common function would be something as simple as multiplying two numbers together.
